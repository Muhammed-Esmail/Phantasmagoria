# Phantasmagoria
 
A real-time multiplayer web game where players collaboratively write chaotic, funny stories. Each person only seeing the last paragraph from the previous writer before continuing the tale.
 
---
 
## What it is
 
Players join a lobby, get assigned a story, and write a segment under a time limit. After each round, stories are shuffled so everyone continues a different story with no full context. At the end, the complete stories are revealed line by line with author attribution.
 
**Key mechanics**
- 3–10 players per lobby, no accounts needed just a username and a 6-digit code
- Each player only sees the last paragraph (or last sentence in Blind Mode) of the story they receive
- 1-minute default timer per turn, auto-submits on expiry
- AI fills in for disconnected players, scores stories with deadpan sanity ratings, and can "Frankenstein" a story into something almost coherent
- Reveal phase is host-controlled, line by line, with live emoji reactions from all players
 
**Game modes**
- Standard — default rules
- Chaos Mode — AI imposes random constraints mid-turn ("your response must include a goat")
- Rapid Rounds — 20-second timer
- Blind Mode — only the last sentence is shown
